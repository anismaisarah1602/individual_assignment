Introduction

JavaScript is a programming language used to create dynamic web pages.

It runs in the browser and allows interactivity.

console.log("Hello World");
  • Client-side scripting
  • Dynamic content
Variables and Data Types

Variables store data values.

You can use var, let, or const.

let name = "John"; const age = 25;
  • String
  • Number
Functions

Functions perform tasks.

They can take inputs and return values.

function greet(){ return "Hi"; }
  • Reusable
  • Modular
Loops

Loops repeat code.

Common loops are for and while.

for(let i=0;i<5;i++){}
  • For loop
  • While loop
Events

Events respond to user actions.

Examples include clicks and typing.

button.onclick = function(){ alert("Clicked"); }
  • Click
  • Keyboard
